# 🎓 Student Management System

A console-based **Student Management System** developed in **Free Pascal** for managing student records, recording marks, generating reports, and ranking students. The system uses text files for persistent data storage and demonstrates core programming concepts such as file handling, searching, sorting, arrays, and input validation.

---

## 📌 Features

- ➕ Add new students
- 📋 View all students
- 🔍 Search students by Registration Number or Name
- ✏️ Update student information
- 🗑️ Delete students with confirmation
- 📝 Record student marks
- 📊 Generate grade reports
- 📄 View individual student transcripts
- 📈 Student statistics
  - Highest score
  - Lowest score
  - Average score
  - Grade distribution
- 🏆 Best student report
- 🥇 Class ranking using Bubble Sort
- ✅ Input validation
- 🚫 Duplicate registration number prevention
- 🚫 Duplicate grade record prevention
